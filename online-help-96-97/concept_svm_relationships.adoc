= SVM 관계
:icons: font
:imagesdir: ../media/


[role="lead"]
SVM(스토리지 가상 시스템) DR(재해 복구)은 SVM의 구성요소 볼륨에 있는 데이터를 복구하고 SVM 구성을 복구하여 SVM 레벨에서 재해 복구 기능을 제공합니다.

System Manager를 사용하여 SVM 간에 미러 관계를 생성하고 관리할 수 있으며, 미러 및 소산 관계를 관리할 수 있습니다.
