= 지연 시간은 얼마인지
:icons: font
:imagesdir: ../media/


[role="lead"]
지연 시간은 대상 시스템이 소스 시스템 뒤에 걸쳐지는 시간입니다.

지연 시간은 현재 시간과 타겟 시스템에 마지막으로 성공적으로 전송된 스냅샷 복사본의 타임 스탬프의 차이입니다. 지연 시간은 소스 및 대상 시스템의 시계가 동기화되지 않는 한 항상 마지막으로 성공한 전송 기간보다 훨씬 오래 지속됩니다. 시간대 차이는 지연 시간으로 자동 계산됩니다.
