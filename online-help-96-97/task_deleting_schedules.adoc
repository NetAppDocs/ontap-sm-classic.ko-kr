= 스케줄을 삭제하는 중입니다
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager를 사용하여 특정 스토리지 관리 작업을 실행하는 스케줄을 삭제할 수 있습니다.

.단계
. Protection * > * Schedules * 를 클릭합니다.
. 삭제할 일정을 선택하고 * 삭제 * 를 클릭합니다.
. 확인 확인란을 선택한 다음 * 삭제 * 를 클릭합니다.
