= 노드 이름 편집
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
필요한 경우 System Manager를 사용하여 노드 이름을 수정할 수 있습니다.

.단계
. 구성 * > * 클러스터 * > * 구성 업데이트 * 를 클릭합니다.
. 노드 * 탭에서 이름을 바꿀 노드를 선택한 다음 * 노드 이름 편집 * 을 클릭합니다.
. 노드 이름 편집 * 대화 상자에서 노드의 새 이름을 입력한 다음 * 제출 * 을 클릭합니다.
