= 역할
:icons: font
:imagesdir: ../media/


[role="lead"]
액세스 제어 역할을 사용하여 시스템에 대한 사용자의 액세스 수준을 제어할 수 있습니다. 미리 정의된 역할을 사용하는 것 외에도 새 액세스 제어 역할을 만들거나, 역할을 수정하거나, 삭제하거나, 역할 사용자에 대한 계정 제한을 지정할 수 있습니다.
