= 이니시에이터 그룹을 삭제합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager의 이니시에이터 그룹 탭을 사용하여 이니시에이터 그룹을 삭제할 수 있습니다.

이니시에이터 그룹에 매핑된 모든 LUN은 수동으로 매핑 해제해야 합니다.

.단계
. 스토리지 * > * LUN * 을 클릭합니다.
. 이니시에이터 그룹 * 탭에서 삭제할 이니시에이터 그룹을 하나 이상 선택한 다음 * 삭제 * 를 클릭합니다.
. 삭제 * 를 클릭합니다.
. 삭제한 이니시에이터 그룹이 * 이니시에이터 그룹 * 탭에 더 이상 표시되지 않는지 확인합니다.
