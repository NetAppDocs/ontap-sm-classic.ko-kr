= 클라우드 계층 삭제
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager를 사용하면 더 이상 필요하지 않은 클라우드 계층을 삭제할 수 있습니다.

클라우드 계층과 연결된 FabricPool 지원 애그리게이트를 삭제해야 합니다.

.단계
. 스토리지 * > * 애그리게이트 및 디스크 * > * 클라우드 계층 * 을 클릭합니다.
. 삭제할 클라우드 계층을 선택한 다음 * 삭제 * 를 클릭합니다.
