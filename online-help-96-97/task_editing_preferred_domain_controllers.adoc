= 기본 도메인 컨트롤러 편집
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager를 사용하여 특정 도메인에 대해 구성된 기본 도메인 컨트롤러의 IP 주소를 수정할 수 있습니다.

.단계
. 스토리지 * > * SVM * 을 클릭합니다.
. SVM을 선택한 다음 * SVM 설정 * 을 클릭합니다.
. 도메인 * 탭의 * 기본 도메인 컨트롤러 * 영역에서 편집할 도메인 컨트롤러를 두 번 클릭합니다.
. 도메인 컨트롤러의 IP 주소를 수정한 다음 * 저장 * 을 클릭합니다.
