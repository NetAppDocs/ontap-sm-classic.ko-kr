= 이더넷 포트 설정 편집
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager를 사용하여 이중 모드 및 속도 설정과 같은 이더넷 포트 설정을 편집할 수 있습니다.

.단계
. 네트워크 * > * 이더넷 포트 * 를 클릭합니다.
. 물리적 포트를 선택한 다음 * 편집 * 을 클릭합니다.
. 이더넷 포트 편집 * 대화 상자에서 양면 인쇄 모드 및 속도 설정을 '수동' 또는 '자동'으로 수정합니다.
. 편집 * 을 클릭합니다.
