= 홈 디렉토리 공유를 생성합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager를 사용하여 홈 디렉토리 공유를 생성하고 홈 디렉토리 검색 경로를 관리할 수 있습니다.

CIFS를 설정하고 시작해야 합니다.

.단계
. 스토리지 * > * 공유 * 를 클릭합니다.
. Create Home Directory * 를 클릭한 다음 사용자가 디렉터리에 매핑되는 방법을 결정하는 패턴 정보를 제공합니다.
. Create * 를 클릭합니다.
. 생성한 홈 디렉토리가 * Shares * 창에 나열되는지 확인합니다.
