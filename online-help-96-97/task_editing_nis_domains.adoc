= NIS 도메인 편집
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager를 사용하면 SVM(스토리지 가상 시스템) 인증 및 인증 요구사항에 따라 NIS 도메인을 수정할 수 있습니다.

.단계
. 스토리지 * > * SVM * 을 클릭합니다.
. SVM을 선택한 다음 * SVM 설정 * 을 클릭합니다.
. 서비스 * 창에서 * NIS * 를 클릭합니다.
. NIS 도메인을 선택한 다음 * 편집 * 을 클릭합니다.
. 필요한 내용을 변경한 다음 * 편집 * 을 클릭합니다.
