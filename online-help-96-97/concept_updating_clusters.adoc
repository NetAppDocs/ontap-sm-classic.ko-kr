= 클러스터 업데이트 중
:icons: font
:imagesdir: ../media/


[role="lead"]
System Manager를 사용하여 고가용성(HA) 쌍의 클러스터 또는 개별 노드를 업데이트할 수 있습니다. MetroCluster 구성에서 클러스터를 업데이트할 수도 있습니다.
